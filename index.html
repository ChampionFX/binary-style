binary-style

The goal of the project is to serve as base style for all Binary projects, and to ensure consistency and maintainability of the brand presentation.

Usage

  • Include jQuery (Version 2.1.1) to your project
  • Include the JS file to your project
  • Include the styles by including this file to your project
